# Персональный сайт учителя математики и информатики

Веб-приложение, разработанное на Flask, предназначенное для публикации методических материалов, разработок уроков, презентаций и других образовательных ресурсов.

## Возможности

- Главная страница
- Информация об учителе
- Методическая копилка
- Материалы по математике
- Материалы по информатике
- Страницы для каждого класса
- Пользовательские страницы ошибок (403, 404, 500)
- Автоматическая генерация sitemap.xml
- robots.txt
- Адаптивный дизайн

## Используемые технологии

- Python 3
- Flask
- HTML5
- CSS3
- Jinja2
- JSON

## Установка

Создать виртуальное окружение:

```bash
python -m venv .venv
```

Активировать его.

Windows:

```bash
.venv\Scripts\activate
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить сайт:

```bash
python app.py
```

После запуска сайт будет доступен по адресу:

```
http://127.0.0.1:5000
```
